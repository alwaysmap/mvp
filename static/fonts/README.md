# Required Fonts

Download the following fonts from Google Fonts and place the TTF files in this directory:

## Cormorant Garamond
https://fonts.google.com/specimen/Cormorant+Garamond

Required files:
- `CormorantGaramond-Regular.ttf` (400)
- `CormorantGaramond-SemiBold.ttf` (600)
- `CormorantGaramond-Bold.ttf` (700)
- `CormorantGaramond-Italic.ttf` (400 italic)

## DM Sans
https://fonts.google.com/specimen/DM+Sans

Required files:
- `DMSans-Regular.ttf` (400)
- `DMSans-Medium.ttf` (500)
- `DMSans-SemiBold.ttf` (600)

## How to Download

1. Visit each font page above
2. Click "Download family"
3. Extract the TTF files from the downloaded ZIP
4. Copy the specific weight TTF files listed above to this directory
